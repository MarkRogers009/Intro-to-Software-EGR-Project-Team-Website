Overview of Project

In this project you will create either a website or an app that provides a guided tour of the campus and shows buildings and locations of interest, such as the tunnel system. The application should provide details, such as what each building is for.

Requirements Interview

  • Covers main academic buildings
  • Give user agency when showing locations
  • Section for student, staff, and visitor
  • Include information and photo about each location
  • Map that can highlight certain buildings
